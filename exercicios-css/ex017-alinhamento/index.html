Alinhamento de textos usando CSS

Este parágrafo está alinhado à esquerda. O alinhamento à esquerda é o padrão para a maioria dos textos em línguas que são lidas da esquerda para a direita.

Este parágrafo está centralizado. O alinhamento central é frequentemente usado para títulos e cabeçalhos, proporcionando um visual equilibrado.

Este parágrafo está alinhado à direita. O alinhamento à direita é menos comum, mas pode ser usado para destacar certos elementos ou em layouts específicos.

Este parágrafo está justificado. O alinhamento justificado ajusta o texto para que ambos os lados fiquem alinhados, criando uma aparência limpa e organizada.
